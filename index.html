How to Play

Your goal is to unlock all the letters of the alphabet (The first two have been unlocked).

  • Click on a letter
  • Click on another letter
  • The combination will show up the top
  • If the combination is successful you will unlock a new letter
  • Keep going to unlock all the letters

Some information

This was a quick weekend game built by Matthew Aisthorpe using knockout.js to get my head around how to build using Model–view–viewmodel (MVVM) software architectural pattern.

The goal was to build a game that could generate itself when restarted to be different everytime.
